# macOS 播放器：IL2CPP 脚本后端

IL2CPP 是一种完全支持的脚本后端，可在为 macOS 播放器构建项目时替代 Mono。

使用 IL2CPP 构建项目时，Unity 会在创建本机二进制文件之前将脚本和程序集内的 IL 代码转换为 C++。请参阅 [IL2CPP](IL2CPP.html) 以了解更多信息。
