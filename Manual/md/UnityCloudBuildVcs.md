# 版本控制系统

为了让 Unity Cloud Build 编译您的项目，必须首先在 [Unity Collaborate](http://docs.unity3d.com/Manual/UnityCollaborate.html) 或第三方源代码控制系统（也称为"版本控制系统"）中托管您的项目。Unity Cloud Build 目前支持：

* [Unity Collaborate](UnityCollaborateEnableCloudBuild.html)

* [Git](UnityCloudBuildVcsGit.html)

* [Apache Subversion (SVN)](UnityCloudBuildVcsSubversion.html)

* [Mercurial](UnityCloudBuildVcsMercurial.html)

* [Perforce](UnityCloudBuildVcsPerforce.html)
