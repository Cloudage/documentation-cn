# XR



XR 是一个总括性术语，包括虚拟现实 (VR)、增强现实 (AR) 和混合现实 (MR) 应用。

本部分将概要介绍 Unity 支持的 VR、AR 和 MR SDK。每个概述部分都包含快速入门指南以及有关故障排除和最佳实践的建议。此外还可以找到有关输入和控制器映射、XR 渲染和音频以及 Unity XR API 的信息。
