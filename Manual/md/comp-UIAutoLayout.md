# 自动布局

[自动布局](UIAutoLayout.html)系统提供了将元素放置在嵌套布局组（如水平组、垂直组或网格）中的方法。使用该系统还可以根据元素包含的内容自动调整元素的大小。

* [内容大小适配器 (Content Size Fitter)](script-ContentSizeFitter.html)
* [布局元素 (Layout Element)](script-LayoutElement.html)
* [水平布局组 (Horizontal Layout Group)](script-HorizontalLayoutGroup.html)
* [垂直布局组 (Vertical Layout Group)](script-VerticalLayoutGroup.html)
* [网格布局组 (Grid Layout Group)](script-GridLayoutGroup.html)
