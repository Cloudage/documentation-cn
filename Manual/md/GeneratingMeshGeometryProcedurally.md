# 程序化网格几何体


[Mesh](../ScriptReference/Mesh.html) 类可让脚本访问对象的网格几何体，允许在运行时创建或修改网格。此方法可用于图形效果（例如，拉伸或挤压对象），但也可用于关卡设计和优化。以下部分将介绍有关如何构建网格的基本详情以及 API 的探索和示例。

* [网格剖析](AnatomyofaMesh.html)。
* [使用 Mesh 类](UsingtheMeshClass.html)。
